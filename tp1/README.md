# TP1 - Nivelación: Creación de un sitio simple

Sitio web con una caja de texto y un botón para cambiar colores, implementado de 3 formas distintas.

**Funcionalidades:**
- Al escribir en la caja de texto y presionar **Enter**, se muestra un alert con el texto ingresado.
- Al hacer click en el botón **"Cambiar Color"**, se alterna el fondo y color del texto entre: blanco/negro, negro/blanco y celeste/rojo.

---

## Consigna 1 - HTML + JavaScript puro

No requiere instalación. Abrir el archivo directamente en el navegador:

```
consigna-1/index.html
```

---

## Consigna 2 - Servidor web con Node.js (Express)

```bash
cd consigna-2
npm install
node servidor.js
```

Abrir en el navegador: http://localhost:3000

---

## Consigna 3 - Servidor web con Python

Requiere tener Python instalado. No necesita dependencias externas.

```bash
cd consigna-3
python servidor.py
```

Abrir en el navegador: http://localhost:5000
